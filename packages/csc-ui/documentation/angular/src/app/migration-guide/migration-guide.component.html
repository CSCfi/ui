Migration Guide

Migrating to CSC-UI 1.0.0

This guide details the changes and how to change your code to migrate to CSC UI version 1.0.0. See the full changelog here.

Breaking changes

c-link

Icon usage is simplified

  • path attribute has been removed
  • icon-fill attribute has been removed
  • icon-after attribute has been removed
  • icon-style attribute has been removed
Before 1.0
After 1.0

c-tag

  • New wrapper component c-tags
    • Handles spaces between the tags without margins defined in the c-tag component
    • c-tag components can be used without the wrapper component with no need to remove the extra margins
  • New attribute block
    • Replaces the functionality of fit attribute prior to 1.0
  • Functionality change with fit attribute
    • Stretches the tags to take all available space in a row instead of each tag being full width
  • New event close
    • Emitted on clicking the close button if the tag has a closeable attribute
    • Closeable tags can now be set active on click
